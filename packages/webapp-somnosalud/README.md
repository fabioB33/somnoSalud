# `@somnosalud/webapp-somnosalud`

App SPA web Next.js para evaluación de trastornos del sueño en 12 pasos.

> **Status (2026-05-07):** package skeleton. Migración del HTML monolítico actual deployado en `paulferrero.github.io/somnosalud/` a Next.js 14 App Router + Tailwind + Supabase planificada para Fase 1.

## Stack

- Next.js 14 App Router
- TypeScript estricto
- Tailwind CSS + shadcn/ui
- Supabase (PostgreSQL + Auth + Storage + RLS multi-tenant)
- Vercel deploy
